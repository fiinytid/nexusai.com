⬡ Setup NEXUS AI — Step by Step

STEP 1 — Install Node.js (wajib!)

Download di: https://nodejs.org → pilih LTS → Install → Restart PC


STEP 2 — Jalankan Bridge

cd [folder NEXUS AI]
npm install ws
node bridge.js

STEP 3 — Install Plugin di Studio

C:\Users\[Username]\AppData\Local\Roblox\Plugins\

Copy NexusAI_Plugin.lua ke folder di atas → Restart Studio


STEP 4 — Aktifkan HTTP di Plugin Management

⚠ Centang HANYA SATU localhost:7778 saja!

Jika ada 2 entri, hapus plugin lama dulu dari Script Explorer.


STEP 5 — Connect

Klik ⚡ CONNECT di panel plugin Studio → status hijau = berhasil!
