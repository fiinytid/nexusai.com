Cara Install Plugin NEXUS AI
1
Download file plugin nexusai_plugin_v5.lua dari link yang disediakan
2
Simpan ke: C:\Users\[NamaKamu]\AppData\Local\Roblox\Plugins\
3
Buka Roblox Studio → Game Settings → Security → aktifkan Allow HTTP Requests
4
Klik tombol NEXUS AI di toolbar Roblox Studio → Klik CONNECT TO CLOUD
5
Status akan berubah menjadi CONNECTED [GEMINI] dan Studio badge di web menjadi hijau
6
Sekarang chat di web ini — AI akan langsung build di place kamu!
