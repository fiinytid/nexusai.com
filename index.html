⬡ How to Install NEXUS AI Plugin

Easiest Way — Toolbox

1. Open Roblox Studio

2. Click View → Toolbox

3. Search: NEXUS AI → Install plugin


⚠ Important: Disable old plugin!

In Plugins → Plugin Management, disable "NEXUS AI ROBLOX" (old one) if present.


Allow HTTP Access

Plugin Management → check ONLY ONE localhost:7778


Connect

Click ⚡ CONNECT in NEXUS AI panel → status turns green → done!
