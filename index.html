⬡ How to Install NEXUS AI Plugin

Via Toolbox (Easiest)

1. Open Roblox Studio → View → Toolbox

2. Search: NEXUS AI → Install


⚠ Important

Disable old plugin if present in Plugin Management.


Manual Install

Copy plugin .lua to:

C:\Users\[You]\AppData\Local\Roblox\Plugins\

Allow HTTP Access

Plugin Management → Check localhost:7778


Start bridge

Terminal: node bridge.js → Click ⚡ CONNECT in Studio
