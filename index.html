⬡ How to Connect NEXUS AI Plugin

STEP 1 — Plugin File

Copy NexusAI_Plugin_v32.lua to:

C:\Users\[Username]\AppData\Local\Roblox\Plugins\

Delete any old .lua plugin files first!


STEP 2 — Allow HTTP in Studio

⚠ Plugins tab → Plugin Management → Check ONLY ONE localhost:7778


STEP 3 — Connect

Click ⚡ CONNECT in the NEXUS AI panel → status turns green → done!
